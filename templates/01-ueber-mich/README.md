# Template 1: Über-mich-Seite  ★☆☆

Eine persönliche Profilseite mit Steckbrief, Hobbys und Links.
**Nur HTML & CSS** – perfekt zum Einsteigen.

## So startest du

1. Öffne den Ordner `01-ueber-mich` in deinem Editor.
2. Installiere die Erweiterung **Live Server** (falls noch nicht da).
3. Klick unten rechts auf **»Go Live«** – deine Seite öffnet sich im Browser.
4. Ändere etwas in `index.html`, **speichere** (Strg+S) und schau zu, wie sich die Seite ändert!

## Welche Dateien gibt es?

- `index.html` – der **Inhalt** deiner Seite (Texte, Listen, Bilder).
- `styles.css` – das **Aussehen** (Farben, Schrift, Abstände).

## Das kannst du jetzt tun

### Leicht ★
- [ ] Tausche den **Namen** und den **Spruch** gegen deine eigenen.
- [ ] Wähle ein anderes **Emoji** für den Avatar (z. B. 🐱 🐲 🦄 ⚽).
- [ ] Fülle den **Steckbrief** mit deinen echten Angaben.
- [ ] Schreibe deine echten **Hobbys** in die Liste.

### Mittel ★★
- [ ] Ändere im `styles.css` die Farbe `--haupt` in deine Lieblingsfarbe (klick auf das kleine Farbquadrat!).
- [ ] Füge eine **neue Karte** hinzu, z. B. »Meine Top 5 Spiele« mit einer Liste.
- [ ] Tausche das Avatar-Emoji gegen ein echtes **Foto**: `<img src="...">` statt des Emojis.
- [ ] Füge zwei weitere **Links** zu Seiten ein, die du magst.

### Schwer ★★★
- [ ] Baue eine **zweite Seite** (`hobbys.html`) und verlinke sie von der Startseite.
- [ ] Ändere den **Verlauf** (`linear-gradient`) im Kopfbereich auf andere Farben.
- [ ] Gib jeder Karte eine **andere Hintergrundfarbe**.

### Bonus ⭐
- [ ] Frag die KI oder den Betreuer: »Wie baue ich einen Knopf, der eine Nachricht zeigt?« – und füge einen ein.
